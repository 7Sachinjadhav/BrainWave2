Optimize the following code snippets to make them more efficient:

Q1: The following code finds the sum of numbers from 1 to n. Optimize it:

total = 0
for i in range(1, n+1):
    total += i
print(total)

Q2: The following code checks if a number is prime. Optimize it:

def is_prime(n):
    if n < 2:
        return False
    for i in range(2, n):
        if n % i == 0:
            return False
    return True
